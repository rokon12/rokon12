<div class="about-pillars">
  <section class="about-pillar">
    <h3>Writing</h3>
    <p>I'm the author of <a href="https://learning.oreilly.com/library/view/modern-concurrency-in/9781098165406/" target="_blank" rel="noopener">Modern Concurrency in Java</a> from O'Reilly Media, covering virtual threads, structured concurrency, and modern JVM concurrency patterns. I also serve as editor for the Java Queue at <a href="https://www.infoq.com/profile/A-N-M-Bazlur-Rahman/" target="_blank" rel="noopener">InfoQ</a>, write regularly at <a href="https://foojay.io/today/author/bazlur-rahman/" target="_blank" rel="noopener">Foojay</a>, and publish <a href="https://bazlur.substack.com/" target="_blank" rel="noopener">The Coding Cafe</a> newsletter.</p>
  </section>

  <section class="about-pillar">
    <h3>Speaking</h3>
    <p>I've spoken at DevNexus, Devoxx Morocco, FOSDEM, ConFoo, GeeCON, JNation, JCon, Copenhagen Developers Festival, and many Java user groups. My talks focus on concurrency, virtual threads, JVM internals, and building AI-powered applications with Java.</p>
  </section>

  <section class="about-pillar">
    <h3>Community</h3>
    <p>I founded the <a href="https://jugbd.org/" target="_blank" rel="noopener">Java User Group in Bangladesh</a> in 2013 and have published <a href="https://www.rokomari.com/book/author/47860/a-n-m-bazlur-rahman" target="_blank" rel="noopener">five Java books in Bengali</a>, helping more developers learn the platform in their own language.</p>
